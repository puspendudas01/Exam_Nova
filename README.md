# 🎓 Exam Nova – Online Exam System

## 📌 Project Overview

Exam Nova is a full-stack **Online Examination System** designed to conduct, manage, and monitor exams efficiently. It provides a seamless interface for students and administrators with secure authentication and real-time exam handling.

---

## 🚀 Features

👨‍🎓 Student
Register & Login
Attempt Exams
Auto-submit on time expiry
View Results
Anti-cheating features (tab switch detection, right-click disable)

👩‍🏫 Teacher
Upload Questions (Subject-wise)
Manage Question Bank
View Student Results

👨‍💼 Admin
Manage Users (Students & Teachers)
Create & Manage Exams
Assign Subjects
Monitor System Activity
View Results & Analytics
---

## 🛠️ Tech Stack

### Frontend

* React.js
* Axios
* HTML, CSS, JavaScript

### Backend

* Spring Boot
* REST APIs
* JWT Authentication

### Database

* PostgreSQL

---

## ⚙️ Installation & Setup

### 🔧 Backend Setup

```bash
cd backend
mvn clean install
mvn spring-boot:run
```

### 💻 Frontend Setup

```bash
cd frontend
npm install
npm start
```
